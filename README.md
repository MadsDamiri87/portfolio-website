# software-madsdamiri-dk

Personal portfolio and CV site for Mads Damiri.

## Structure

```txt
src/
  assets/images/      Image assets used by the site
  components/layout/  Page shell and footer
  components/navigation/
  components/projects/
  components/sections/
  components/ui/      Small shared UI components
  data/               Portfolio content
  styles/             Global styles
  types/              Shared TypeScript types
  utils/              Small helpers
```

## Commands

```bash
npm install
npm run dev
npm run build
```
